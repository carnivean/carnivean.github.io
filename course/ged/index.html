Assignment 1

2DSmoothArray

  • Wenn ihr malloc benutzen wollt für das Array, müsst Ihr #include "stdlib.h" hinzufügen
  • smoothArray kann zum Beispiel über eine 4-fach verschachtelte Loops gelöst werden
  • smoothArray stellt sicher, dass eure Indizes für x und y > 0 aber x < width und y < height sind
  • rand() gibt einen int zurück, ihr müsst den Wert in einen float verwandeln um zufällige Werte zwischen 0 und 1 zu erhalten
